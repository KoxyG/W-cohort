#### Getting Started:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/koxyG/W-cohort
   cd directory
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run the App:**
   ```bash
   npm run dev
   ```

   The app will be available at [http://localhost:3000](http://localhost:3000).
